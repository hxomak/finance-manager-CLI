# Documentation on How to run

### Prerequisites
- Python 3.8+
- pip or uv
### Installation
 
```bash
# Clone the repo
git clone <https://github.com/hxomak/finance-manager-CLI.git>
cd <finance-manager-CLI>
 
# Create virtual environment
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
```
 
---
 
## Running the Code
 
### Basic Execution
 
```bash
python src/main.py
```
